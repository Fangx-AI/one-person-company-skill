# Examples

这些示例不是为了展示“回答更会说”，而是展示一人公司.skill 如何把一个想法压回商业现实。

每个示例都包含：

- 用户输入
- 普通 AI 容易给出的弱回答
- 一人公司.skill 应该给出的商业判断卡
- 为什么这个回答更好
- 可参考的市场模式
