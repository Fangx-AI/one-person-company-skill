## 一人公司.skill 商业判断卡

结论：不能直接判死刑，但不要把“能转格式”当成生意。

付费可能性：中低。通用转换器很弱，嵌入专业发布流程才有机会。

最可能付费的人：技术写作者、产品文档团队、内容团队、需要批量转换的开发者或运营团队。

直接竞品：Marked 2、Markdown Monster、CloudConvert、ConvertAPI。

替代方案：Pandoc、markdown-it、remark、VS Code 插件、文档站、CMS、邮件编辑器、帮助中心工具。

可收费切口：稳定样式、代码高亮、批量处理、API 自动化、品牌化帮助中心页面、隐私或本地处理。

第一单动作：不要先做 SaaS。先卖“10 篇 Markdown 文档转成品牌化帮助中心页面”，收 199/499 元服务费。

7 天验证：找 15 个正在写文档或发内容的人，要求他们提供真实 Markdown 文件，并尝试收款。

停损线：15 人里没有 3 个给真实文件并愿意付费，就不要做通用转换器。

## 为什么更好

- 它承认转换工具可以收费，但把付费场景拆清楚。
- 它区分直接竞品、免费替代和相邻替代。
- 它把产品从“功能”收窄成具体可交付结果。
