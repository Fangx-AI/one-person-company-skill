# 技术人不会社交销售

## 用户输入

```text
/opc 我有技术能力，但不会社交和销售，怎么做一人公司？
```

## 普通 AI 容易回答

```text
可以学习销售，打造个人品牌，多交流，坚持输出。
```

## 一人公司.skill 商业判断卡

结论：你的短板不是技术，是没有可重复触达付费用户的分发路径。

付费可能性：取决于你能不能找到公开抱怨、愿意花钱省时间的窄工作流。

最可能付费的人：开发者、运营、SaaS 小团队、需要自动化或模板的垂直人群。

直接竞品：VS Code 插件、Chrome 插件、开发者模板、API、小型 B2B 工具。

替代方案：开源项目、脚本、内部工具、外包自动化、教程。

可收费切口：插件、模板、API、自动化脚本、垂直小工具，优先选择用户已经在公开提问的重复问题。

第一单动作：今天在 GitHub issues、V2EX、即刻、微信群找 30 条重复抱怨，整理成一个 99 元预售承诺。

7 天验证：向 10 个目标用户预售，至少 2 个愿意付费或提供真实文件/数据。

停损线：没人愿意付 99 元，也没人愿意给真实数据，就不要继续开发，换工作流。

## 为什么更好

- 它没有把“不会社交”当成人格缺陷。
- 它把销售问题转成分发路径设计。
- 它给技术人更适合的产品形态。
